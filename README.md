# Student Grade Book (C++)

## Internship Project
This project was developed as part of the **Hex Softwares Internship Program**.

## Project Description
The Student Grade Book is a console-based C++ application that allows users to manage student records and grades.  
It provides a simple menu-driven interface for adding students, recording grades, displaying student information, and calculating averages.

## Features
- Add new students
- Add grades for students
- Display student grades
- Calculate average grades
- Save and load data using file handling

## Technologies Used
- C++
- STL (Standard Template Library)
- File Handling
